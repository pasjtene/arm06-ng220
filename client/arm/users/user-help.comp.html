Help managing users

Creating a user

  1. Click the create new user button
  2. Enter valid user details
  3. Click save. Any error will be reported
  4. if there is no Error, the navigation pan will close and the user can be seen in the list

Deleting a user

  1. Hover the mouse over the user you want to delete
  2. A recycle bin icon will apear on the same line:
  3. Click the recycle bin icon, then confirm or cancel the deletion
