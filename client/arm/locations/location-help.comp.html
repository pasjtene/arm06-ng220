Help managing locations

Adding a new location

  1. Click the create new location button
  2. Enter valid location details
  3. Click save. Any error will be reported
  4. if there is no Error, the navigation pan will close and the location can be seen in the list of locations

Deleting a location

  1. Hover the mouse over the location you want to delete
  2. A recycle bin icon will apear on the same line:
  3. Click the recycle bin icon, then confirm or cancel the deletion
